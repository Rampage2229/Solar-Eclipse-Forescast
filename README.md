# Solar-Eclipse-Forescast
A simple python script that uses data from the NASA Horizons System database to predict solar eclipses of any year (past or future!).

## Installation
To get started, clone the repository and install the necessary libraries. You can simply execute
```pip install requirements.txt -r```
on your python enviroment.

## Running
Simply run ```python Eclipse_Forecast.py``` and follow the script instructions
